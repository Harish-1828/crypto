Diffie-Hellman Key Exchange

📚 How it works:

  • P = A large prime number (public)
  • G = A primitive root of P (public)
  • a = Alice's private key (secret)
  • b = Bob's private key (secret)

Both Alice and Bob will compute the same shared secret without ever transmitting it!
